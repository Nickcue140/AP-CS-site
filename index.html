Drawing App

This is my drawing app

This is my drawing app. It is not very advanced, but it has some features that I think are very interesting. I added a glide to button, which when combined with the draw shape you can make those squiggly lines or when combined with space can make textured lines.

Farming game

My summer homework

For my scratch project I chose to make a farming game. You are able to plant corn by pressing space and when you get enough you can go to the house to win. My goal was to make the corn grow over time, then be able to collect it. To accomplish this I used a lot of wait blocks. This allowed the code to stop for a bit to pretend like the corn was taking time to grow.
